# Netflix Clone

A simple Netflix clone that includes the core features of Netflix

## Dependencies

- react-router-dom: ^5.2.0
- @splidejs/react-splide: ^0.5.3
- axios: ^0.24.0
- redux: ^7.2.3
- redux-thunk: ^2.4.0
- react-redux: ^7.2.6
